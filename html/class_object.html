Detailed Description

The most basic state of class that can be exist.

This class is the highest base class. All object that exists in system will inherite this class. Object class itself has no rendering feature. However, it will have position, boundingbox and transform matrixes to exists as an virtual object in the scene. (like invisible tracking node) To actually render the object, you will have to use RenderableObject.
